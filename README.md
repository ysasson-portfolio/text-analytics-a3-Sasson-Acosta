## Overview
This assignment compares two pretrained topic modeling methodologies (LDA with CountVectorizer and NMF with TF-IDF) 
on Disneyland Reviews. The goal is to understand strengths and weaknesses of each approach.

## Dataset
- **Name:** Disneyland Reviews
- **Source:** https://www.kaggle.com/datasets/arushchillar/disneyland-reviews
- **Size:** 42,656 rows
- **Domain:** Amusement Park Reviews, Topic Modeling, slight Sentiment Analysis (mainly for coming up with the topic names.

- `notebooks/01_eda.ipynb` - Exploratory Data analysis notebook
- `notebooks/02_eda.ipynb` - Pre-Processing the Dataset notebook
- `notebooks/03_LDA.ipynb` - Topic Modeling through LDA methodology notebook
- `notebooks/04_NMF_TFIDF.ipynb` - Topic Modeling through NMF methodology notebook
- `notebooks/05_Interpretation.ipynb` - Interpretation of the topic models give to us by the model
